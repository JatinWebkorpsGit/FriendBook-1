What is FriendBook?

Secure Signup & Login

Users can create an account and log in securely using Spring Security with password encryption.

User Profile

Update profile picture, view own posts, followers/following count, and edit profile details easily.

Follow / Unfollow Users

Users can follow or unfollow others, and view follower/following counts dynamically.

Create, Edit, Delete Posts

Post images with captions, edit them, and delete your own posts. All posts are displayed in the main feed.

Like / Unlike Posts

Users can like or unlike any post, and the like count updates in real-time.

Comment on Posts

Just like Instagram, users can leave comments under posts, and they are visible instantly below each post.

Live News Feed

Explore all posts from users across FriendBook in a clean and modern feed.
